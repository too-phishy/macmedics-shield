## To install dependencies:

```
nvm use 21
npm install
```

## To run the entire test suite:

```
npm test
```

## Run linter:

```
npm run prettier
```

## Send pull requests

Issues are cool, but PRs are better.

If you add features, add tests. Don't break the tests.
